## Lancement et contrôles du programme
- se mettre dans TD07_Train
- run le programme avec "../bin/TD07_Trainex01.exe" "circuit/circuit.json"
- penser à rebuild après chaque modification du programme

## Contrôles
*(Les controles sont en AZERTY mais la bibliotheque etant en QWERTY nous avons adapté les controles)*
Déplacements :
- Z=avant
- Q=Gauche
- S=Arriere
- D=Droite
- Space=haut
- W=Bas
Modes : 
- L=activer la lumière
- N=switch entre jour et nuit
- F=mode filaire
- P=mode plein
Autres :
- ESC=quitte le programme

les *fleches directionnelles* servent à modifier l'angle de la caméra
*(nb: Nous avons utilisé 'W' pour aller en bas car shift ou controle fonctionnait mais pas si on reste appuyer étrangement)*
